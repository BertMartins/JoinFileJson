FileJson

						
Este sistema de mesclagem de arquivos oferece uma solução fácil e eficiente para combinar e manipular múltiplos arquivos de projeto JSON em um único arquivo. Ele foi desenvolvido para facilitar o gerenciamento e a consolidação de dados em projetos que utilizam arquivos JSON, especialmente aqueles que lidam com testes e suites.
